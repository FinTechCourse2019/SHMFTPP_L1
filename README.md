#### 金融
经济活动的高阶抽象，资金的融通（跨时间跨地域跨市场，根源于共识的想象力）
1. 今天的1块钱不等于明天的1块钱 --- 货币的时间价值    
2. 不要把鸡蛋放到同一个篮子里  --- 分散风险   
3. 天下没有免费的午餐 ---- 贴现率

把不易流动的“化整为零” -> 标准化合约，形成繁荣的二级市场    
运用已有的信用，用合适的杠杆，守住风险底线    
底层资产包装太多（如同code函数多层嵌套），脆弱环节暴雷，所有相关方都血残玩不下去    

可参考黄奇帆的关于金融的论述
